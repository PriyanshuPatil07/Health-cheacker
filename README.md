# Health-cheacker
This Python program collects basic symptoms like temperature, cough, headache, and tiredness. It summarizes the user’s inputs and gives simple comfort tips such as rest and hydration. It is friendly, easy to use, and meant for learning, not diagnosis.
Features of the Project

User-friendly interface with simple questions.

Takes temperature input and checks if it is normal or high.

Collects basic symptoms like cough, headache, and tiredness.

Summarizes all symptoms clearly for the user.

Provides general comfort tips such as rest, warm fluids, and hydration.

Non-medical and safe, only gives basic wellness suggestions.

Uses simple Python logic, easy for beginners to understand

Lightweight and fast, runs instantly on any Python environment.

Can be extended easily to include more symptoms or features.

Technology and Tools Used

Programming Language: Python 3

Development Environment: Any Python IDE or editor
(e.g., VS Code, PyCharm, Thonny, IDLE, Jupyter Notebook)

Input/Output Functions: input(), print()

Control Structures: if, elif, else

String Handling: .lower() for processing user responses

Basic Data Types: float, str, char (via string input)

Console-based Interface: Text-based user interaction

Steps to Install and Run the Project
1. Install Python

Go to the official Python website.

Download Python 3.x for your operating system (Windows/Mac/Linux).

Install Python and make sure to check “Add Python to PATH” (on Windows).

2. Set Up a Code Editor (Optional but Recommended)

You can use any editor such as:

VS Code

PyCharm

Thonny

IDLE (comes with Python)

Jupyter Notebook

3. Create the Python File

Open your editor.

Create a new file named:
symptom_checker.py

Copy the program code into the file.

Save the File

Save the file in any folder you choose.

5. Run the Program

You can run the program in two ways:

Run from the Editor

Open the file in your editor.

Click Run or Play button.

The console will open and ask for inputs.
